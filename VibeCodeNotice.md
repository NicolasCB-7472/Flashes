# VibeCode Notice

## Qué es Flashes

**Flashes** es una aplicación de escritorio de tarjetas de estudio (flashcards)
construida en Java con Swing. Permite al usuario importar barajas en formato CSV,
repasar tarjetas con animación de volteo, marcar respuestas como correctas o
incorrectas, y revisar el historial de sesiones mediante un sistema de logs.
Incluye perfil de usuario con avatares y opción de orden aleatorio o secuencial.
